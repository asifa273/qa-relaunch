
QA Relaunch Portfolio
This repository is a collection of projects focusing on modern Quality Assurance, Software Engineering, and AI-driven automation. It serves as a comprehensive resource for learning and demonstrating skills in automated testing, data validation, and development.

📁 Projects Overview
1. Selenium - LoginFlowAutomator
Purpose: Automated test scripts for verifying user authentication and product sorting on web platforms (e.g., SauceDemo).

How to Run: 1. Navigate to Selenium/LoginFlowAutomator.
2. Run: mvn test

2. Python Learning Project
Purpose: Covers core Python concepts, data manipulation, and basic automation testing using the Pytest framework.

How to Run: 1. Install requirements: pip install -r python/requirements.txt
2. Run tests: pytest python/pytest/basicTests/

3. SQL QA Portfolio
Purpose: Focuses on data quality assurance, including identifying nulls, detecting duplicates, and data reconciliation in financial/regulatory contexts.

How to Run: Execute the .sql scripts in any standard SQL environment (PostgreSQL/SQLite) or via db-fiddle.com.

4. AI Defect Resolver
Purpose: A tool integrating the Claude AI API to analyze bug reports and propose or execute code fixes automatically.

How to Run: 1. Set your ANTHROPIC_API_KEY.
2. Install requirements in the ai/ folder.
3. Run the resolver script.

5. Java Exercises
Purpose: Core Java logic and Object-Oriented Programming (OOP) exercises designed for building a strong coding foundation.

How to Run: Compile using javac FileName.java and run with java FileName.

6. Postman Collections
Purpose: API testing collections for verifying REST endpoints, status codes, and JSON response bodies.

How to Run: Import the .json collection files into the Postman desktop client.
